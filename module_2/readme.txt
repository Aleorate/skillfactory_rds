Основные цели и задачи проекта:
Провести разведывательный анализ данных и составьте отчёт по его результатам, в частности,
проанализировать какие параметры влияют на итоговую оценку учащегося на экзамене по поматематике, 
на основе полученной информации составить таблицу только с корреляционно значимыми данными.

Краткая информация о данных:
В таблице представлены 30 переменных, которые имеют непосредственное отношение к жизни учащегося. Они относятся к разлиным сферам его жизни: личной жизни, семье, учебе, его условиях проживания.

Этапы работы над проектом:
Провел первичную обработку данных.
Работа со столбцами: удалил повторяющиеся, переименовал для большего удобства оставшиеся.
Оценил количество уникальных значений для номинативных переменных, исправил ошибки,
предобразовал недемонстративные данные в более удобный для восприятия формат.
Посмотрел на распределение признака для числовых переменных, устранил выбросы, где это необходимо.
Построил графики, которые наглядно отражают корреляцию значений.
Провел корреляционный анализ количественных и номинативных переменных и устранил те, которые не влияют на предсказываемую величину.
Построил таблицу для корреляционно значимых данных и сформулировал выводы.
